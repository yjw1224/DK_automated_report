# DK_automated_report
모바일 환경에서 수기로 최신화하기에는 오타와 양식 깨짐 가능성이 매우 높아지는 대표병 보고를 시스템화하여
오류를 최소화시키는 데에 목적이 있음.


## 개발 툴
HTML / TailwindCSS / Svelte + Vite
SPA 형식으로 개발.

## 주요 기능
### 메인 화면
무슨 포대 몇 생활관인지 작성할 수 있음. (1포대 / 2포대 / 3포대 / 본부포대)
오늘의 날짜를 선택 후 시작할 수 있음.
아래의 세 페이지로 이동할 수 있는 버튼.

### 생활관 인원 관리 페이지 (PERSONNEL)
생활관별 세로 2x5 배치를 적용.
최대 10명의 인원 추가 가능.
각 자리를 클릭 시 신규 인원을 추가할 수 있음. (이미 선택된 자리는 변경 또는 삭제 가능)
총원 N 열외 M 열외내용 [근무 1, 휴가 1, ...] 을 제외한 현재원 N-M.

#### __신규 인원 추가__
__[인원 특성]__
```
0. [계급] + ' ' + [이름]
[계급]은 이병 / 일병 / 상병 / 병장 네 종류가 있다.
[이름]은 인원의 이름.
이 두 특성을 조합하여 [관등성명]이 완성된다.

1. 열외 여부 (Y/N) + [열외 사유]
[열외 사유]
근무 / 휴가 / 외출 / 외박 / 상황병 / 입실 / 외진
그 외의 커스텀 사유는 직접 입력하여 추가 가능.

[개별 설정]
2. 출타 (휴가 / 평일외출 / 주말외출 / 외박)
- 휴가 : 시작일/종료일이 존재. 둘은 서로 겹칠 수 없다.
- 평일외출 : 평일 하루에 나가는 외출
- 주말외출 : 주말 하루에 나가는 외출
- 외박 : 주말 및 공휴일 1박 2일 동안 나가는 외출. (공휴일이 있으므로 꼭 주말에만 나가는 건 아님)


3. 외진 (외진 여부 / 외진 날짜 / 외진 장소)
4. 면회 (면회 여부 / 날짜 / 면회자)

[단체 설정]
5. 민간 이발 (Y/N)
6. 종교 (기독교 / 천주교 / 불교)
7. 병기본 (체력 측정 / 화생방 / TCCC / 정신전력 / 사격)
8. 배달 음식 [(날짜 / 종류 / 인원들)로 이루어진 데이터 집합들]
```

#### __새로운 [인원 특성]을 추가하고 싶을 때__
인원 특성은 다음과 같은 데이터로 정의됨.
```
1. 특성 이름
2. 소분류 (예를 들어, 5번 병기본인 경우 네 가지의 선택지)
3. 소분류별 그에 해당하는 인원
```

### 대표병 보고 페이지 (MESSAGE)
크게 [총원 및 열외, 특성, 그 외 사항]으로 나뉨.
아래의 세 가지 사항은 가장 최근에 작성했던 날짜로부터 최신화함. (초기화되지 않도록)

#### __총원 및 열외__
생활관 인원 관리에서의 변경사항 저장.
여기에서도 인원별로 열외 여부와 열외 사유를 변경할 수 있도록 함.

#### __특성__
위의 [인원 특성] 및 새로 추가한 커스텀 특성을 모두 나열함.
특성의 추가 / 수정 / 변경, 소분류별 인원 수정이 모두 가능함.

#### __그 외 사항__
```
1. 구타/가혹행위를 당하고 있는 인원
2. 분대원 면담 및 관찰 결과
3. 그 외 특이사항
```
세 가지 사항 모두 있음 / 없음 상태로 구분. 있다면 세부 내용을 적을 수 있도록 함.

위의 세 가지 사항을 잘 정리된 하나의 [메시지 형식]으로 정리.

[메시지 형식] - 예시.

```
본부포대 3생활관 
총원 7 현재원 7입니다.

🏠 출타 

2/25~3/4 일병 홍길동 휴가 예정입니다.

🏥 외진

2/23 상병 이리듐 고양병원 외진 예정입니다. 

💈 민간이발

상병 이리듐, 일병 홍길동 희망합니다.

⬆️ 병기본

🔫 사격
상병 이리듐, 일병 홍길순, 홍길동 희망합니다.

🩹 tccc
일병 홍길순 희망합니다.

🥽 화생방
일병 홍길순 희망합니다.

📖 정신전력
일병 로렌슘, 홍길순 희망합니다.

🏃 체력측정
상병 나트륨, 이리듐 희망합니다.
```


```
본부포대 3생활관 
총원 5 열외 2
열외내용 당직 1 근무 1을 제외한 현재원 3입니다.

🏠 출타 

-

⛪️ 종교

[불교]
상병 칸예웨스트, 일병 켄드릭라마, 드레이크, 이병 릴테카 희망합니다.

🏥 외진

-

✂️ 민간이발
상병 칸예웨스트, 에미넴 희망합니다.

🪖 병기본

[정신전력]
상병 칸예웨스트, 일병 켄드릭라마, 이병 릴테카 희망합니다.

[체력측정]
상병 칸예웨스트, 일병 켄드릭라마, 이병 릴테카 희망합니다.

[TCCC]
상병 칸예웨스트, 일병 켄드릭라마, 이병 릴테카 희망합니다.

[화생방]
상병 칸예웨스트, 일병 켄드릭라마, 이병 릴테카 희망합니다.
```


```
본부포대 1생활관 총원 8 열외 4 열외내용 휴가 4을 제외한 현재원 4입니다.

상병 켄드릭라마 2/12 ~ 2/24 휴가 중입니다.

병장 드레이크 2/15 ~ 2/24 휴가 중입니다.

병장 스눕독 2/21 ~ 2/24 휴가 중입니다.

병장 칸예웨스트 2/23~3/4 휴가 중입니다. 

2/28 상병 켄드릭라마 면회 (부모님, 형, 동생) 희망합니다.

1생활관 병기본 희망자 없습니다.
```

[저장 및 보내기] 버튼을 누르면 채팅방에 바로 공유될 수 있도록 함.

### 정보 불러오기 페이지 (LOAD)
위의 보고 페이지에서 최종 정리 완료된 양식 메시지를 입력하면
자동으로 인원/특성 정보를 불러올 수 있는 페이지.
대표병 인수인계 시 필요하다.
